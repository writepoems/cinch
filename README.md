# cinch

Dead simple to-do app. Pretty generic, but it's simple.

## Why?

I needed something to keep track of my tasks, and I didn't want something bloated that would make me create an account and had so many buttons and UI elements everywhere.

My solution? Building a simple app with React in an hour or two, letting people keep track of tasks while saving them to your browser locally. No bloat, no sign-ups, and no gimmicks.

I might update this every now and then to better suite my future needs, as well as add some settings (and possibly mobile layout support, as well as task deletion once I figure out an easy way to let the user do it without an extra button or any additional UI components).
