# cinch

Dead simple to-do app. Pretty generic, but it's simple.

## Why

When I had fell back on my homework and had many missing assignments overdue, I wanted a way for me to track them since my school's SIS (Student Information System) was extremely outdated and lacked literally *any* modern functionality.

I didn't want to use any bloated to-do websites, primarily because they all require accounts, have ads and useless things, or just don't fit my vibe (yes, I'm very picky about design and looks).

My solution? Make my own of course. No ads, features I'd use, and absolutely no bloat. Only things I would enjoy the most, and the simplest UI ever.

## Setup

**Prerequisites**
- Git
- Bun or Node.js <sub>that's literally it, super simple</sub>

If you want to setup cinch to either contribute or just change it to match your needs, you can run the following commands to get started:

1. Clone the repository
```sh
git clone https://github.com/writepoems/cinch.git cinch
```

2. Install dependencies
```sh
bun i # if you're using node, do "npm i" instead
```

3. Run the development server
```sh
bun dev # or "npm run dev"
```
